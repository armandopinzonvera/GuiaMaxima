Despues de tener todas las tablas, para crear la relacion, entramos a la tabla que va a tener el enlace, y a la pestaña Estructura y hacemos clic en Vista de relaciones

A lo que se abre una nueva ventana, donde colocamos nombre a la relacion, en el primer EditText, en la casilla de Columna colocamos el campo de la tabla tiene la llave foranea (donde se hace la relacion), en Base de datos el nombre de la BBDD a la que pertenece, luego la tabla que se llama Tabla y en Columna la primary Key de esa tabla.
en las casillas de ON DELETE y ON UPDATE colocamos en cascada, para que las actualizaciones se apliquen a todas las tablas anteriores.

Para que el usuario entre se le envía el enlace URL de la cuenta

Para configurar MFA (Autenticación multifactor), con lo cual se agrega un mecanismo adicional de verificación, a través de otro dispositivo hardware generador de códigos, seleccionamos el que permite emplear el celular: Dispositivo MFA virtual, observamos el listado de aplicaciones compatibles para leer el código QR y generar la autenticación: Authy, Duo Mobile, LastPass Authenticator, Microsoft Authenticator, Google Authenticator; escogemos la última y desde nuestros celular escaneamos el código.
