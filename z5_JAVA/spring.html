SPRING

Instalacion

Spring es un framework para Java, es decir un entorno de trabajo, para facilitar el desarrollo de aplicaciones; es la alternativa de aplicaciones para JEE (Java enterprice).
Dentro de sus ventajas esta la inyección de dependencias, favoreciendo el loose coupling(hacer modificaciones a múltiples clases), además el desarrollo sencillo con POJOs(Plain old Java Objects), también minimiza el código repetitivo, simplifica el acceso a datos, y la programación orientadas a Aspectos. tiene como requisitos: tener el JDK, JRE versión >=8, servidor java(tom cat), IDE(entorno de desarrollo integrado)JAVA. El IDE utilizado debe ser Eclipse EE (Eclipse enterprice edition), y tenerinstalado TomCat como servidor. Procedemos entonces a descargar las Sprin jar files para posteriormenteimportarlas a eclipse

Se descarga la última versión de los spring files que están hasta el final de la hoja;

Luego hacemos click en el primer vinculo, con el que se descarga un archivo (spring-framework-5.2.9.RELEASE) .zip, que posteriormente vamos a extraer

Integrar libreria

Nos colocamos en la vista de Java (window/ perspective/ open perspective) iniciamos un nuevo proyecto: File / new / new Java project, en la ventana que se abre dejamos seleccionado el boton de "use a project specific JRE" y damos en Finish

Ahora el proyecto creado aparece en el listado de carpetas, en el costado izquierdo de la interfaz, damos clic derecho sobre este: build path / configure Build path.

En la ventana que se abre, vamos a la pestaña Order and Export, Colocamos de primero: la carpeta SRC, utilizando los el Boton lateral

Ahora, vamos a la pestaña Libraries, damos un click sobre Modulepath (tiene dos flechitas) A lo cual se habilitan el boton de Add JARs...

Damos click en el Boton Add External JARs..., A lo cual se abre una nueva ventana, donde buscamos la ruta de la carpeta en donde se encuentra el Spring JAR filesque descargamos, en la carpeta libs; solamente seleccionamos los archivos binarios, es decir que no incluimos los -javadoc & -source

Despues de abrir los archivos seleccionados, damos en Apply y despues en Apply and Close

Con lo cual se nos crea un archivo nuevo llamado: JRE System Library
