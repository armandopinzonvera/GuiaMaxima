RecyclerView

Estructura

Lo primero es incluir en el build.grade las dependecias para el Recycler-View: por la ruta: File/Project Structure/Dependencies, selecionamos +/Library Dependency Colocando las siguientes:
androidx.recyclerview:recyclerview:1.2.0-alpha06 androidx.recyclerview:recyclerview-selection:1.1.0

En el .xml de la GUI, cambiamos el Layout principal a LinearLayot con orientacion vertical. alli agregamos el componente recycler, que ya se genera por la importacion de la libreria:androidx.recyclerview.widget.RecyclerView y le colocamos un ID: "@+id/recyclerId"

Creamos un Elemento en la carpeta Layout = Resource file el cual es un nuevo .xml que contendra lo que aparece en el recycler

Creamos una nueva clase que sera el adaptador para cargar la informacion al "Resource file", a la cual llamaremos Adapter- datos hacemos extends a RecyclerView.Adapter<>
public class AdaptadorRecycler extends RecyclerView.Adapter<AdaptadorRecycler.ViewHolderDatos> {
como nos aparece en rojo la palabra ViewHolderDatos, la declaramos como una subclase dentro de nuestra clase principal (public class ViewHolderDatos{)
Como la clase principal marca error, se le da click en el bonbillo e indicamos implementar metodos y nos implementa los metodos: onCreateViewHolder(), onBindViewHolder() y getItemCount().
Ahora marca error en la declaracion de la clase principal, nos colocamos sobre la señal y aceptamos la recomendacion: Make ViewHolderDatos Extends... indicando que la subclase debe hacer extends, quedando asi:
public class ViewHolderDatos extends RecyclerView.ViewHolder {
luego marca error para esa subclase, indicando que se cree el constructor y aceptamos. ya quedando la clase completa.

Dentro de esa clase, generamos una referencia a la lista que se va a generar, mediante una variable general ArrayList<String>listDatos; , ademas generamos un constructor con esta variable

Seguimos en esa clase y ahora, en la subclase ViewHolderDatos generamos una referencia al componente (ó todos los componentes que se tengan) en Resource file (item_list_recycler.xml); con Componente, me refiero a TextView, EditText .. y todos esos elementos que estan dentro del .xml; que hasta ahora solo hay un editText con id: tv_datorecycler.
La referencia se hace llamando esos componentes con findViewById(R.id.--), como se hace tradicionalmente entre el archivo .java y .xml, pero incluyendo el objeto View del constructor de esta subclase itemView, quedando de la siguiente manera:
tv_datorecycler = itemView.findViewById(R.id.tv_datorecycler);

Seguimos en la misma clase, pero ahora vamos al metodo onCreateViewHolder(), el cual enlaza esta clase - adaptador con el Resource file (item_list_recycler.xml), entonces dentro de el, Creamos un objeto de la clase View que inicialice como LayoutInflater:
View view = LayoutInflater.from(parent.getContext()).inflate(R.layout.item_list_recycler, null, false); Ademas, retornando: new ViewHolderDatos(view)

Seguimos en la misma clase, pero ahora vamos al metodo getItemCount(), el cual retorna el tamaño de la lista de datos, entonces le colocamos en el return: la variable ArrayList, que habiamos creado, con el metodo .size():
return ListDatos.size();
Ahora vamos al metodo onBindViewHolder(), que se encarga dela comunicacion entre esta clase-adaptador y la subclase ViewHolderDatos, que esta adentro. Entonces le colocamos el objeto holder y escribimos un metodo llamado asignarDatos() que aun no existe(indicaremos a AS, que lo cree dentro de la subclase ViewHolderDatos), con parametros: (listDatos.get(position)).

En este metodo recien creado, asignarDatos(String s), le indicamos que coja la referencia al elemento TextView que habiamos hecho previamente y le asigne contenido, mediante el metodo setText() con el String "s" que esta como parametro del metodo. 

                           import android.view.LayoutInflater;
                            import android.view.View;
                            import android.view.ViewGroup;
                            import android.widget.TextView;
                            import androidx.annotation.NonNull;
                            import androidx.recyclerview.widget.RecyclerView;
                            import java.util.ArrayList;
                            
                            public class AdaptadorRecycler extends RecyclerView.Adapter {
                            
                                ArrayList listDatos;
                            
                                public AdaptadorRecycler(ArrayList listDatos) {
                                    this.listDatos = listDatos;
                                }
                            
                                @NonNull
                                @Override
                                public ViewHolderDatos onCreateViewHolder(@NonNull ViewGroup parent, int viewType) {
                            
                                    View view = LayoutInflater.from(parent.getContext())
                                            .inflate(R.layout.item_list_recycler, null, false);
                            
                                    return new ViewHolderDatos(view);
                                }
                                @Override
                                public void onBindViewHolder(@NonNull ViewHolderDatos holder, int position) {
                                    holder.asignarDatos(listDatos.get(position));
                                }
                                
                                @Override
                                public int getItemCount() {
                                    return listDatos.size();
                                }
                            
                                public class ViewHolderDatos extends RecyclerView.ViewHolder {
                            
                                    TextView tv_datorecycler;
                            
                                    public ViewHolderDatos(@NonNull View itemView) {
                                        super(itemView);
                                        tv_datorecycler = itemView.findViewById(R.id.tv_datorecycler);
                                    }
                                    public void asignarDatos(String datos) {
                                        tv_datorecycler.setText(datos);
                                    }
                                }
                            }

Dejamos listo el adaptador, nos dirijimos entonces a la clase del GUI que para este ejemplo es class Continuar, en donde colocamos dos elementos:
una lista igual al de la clase adaptador: ArrayList<String> listDatos; y una referencia a al <androidx.recyclerview.widget.RecyclerView /> , que se creo en el .xml, de esta misma clase.
Este ultimo como un objeto RecyclerView recyclerView;
luego lo instanciamos dentro del metodo onCreate(), enlazando con este elemento, usando el metodo findViewById, y le agregamos el metodo setLayoutManager()
recyclerView = findViewById(R.id.recyclerId);
recyclerView.setLayoutManager(new LinearLayoutManager(this));
Seguimos en el onCreate, y debajo de lo anterior, instanciamos listDatos
listDatos = new ArrayList<String>();
y colocamos un pequeño bucle for para probar el adecuado despliegue, en donde la lista se envia al adaptador para que inicie todo el proceso, instanciando el adaptador
AdaptadorRecycler adaptadorRecycler = new AdaptadorRecycler(listDatos);
recyclerView.setAdapter(adaptadorRecycler);

Fragment Mapa

Para Facilitar el trabajo se crea un Google Maps Fragment que ya trae incorporadas la vinculos y los metodos para mostrar el mapa; en la carpeta del proyecto CD/new/Fragment/Google Maps Fragment.
Faltando solo hacer los vinculos descritos en el apartado anterior, agregar la dependencia en el build.gradle y agregar la API KEY en el Manifest.
Para obtener el numero, se sigue el proceso descrito en el capitulo de Android / Map-Location de esta guia.
Para agregar la dependencia de google-play-service-maps en Project Structure File/Project Structure/Dependencies, selecionamos +/Library Dependency
a lo que se abre una nueva ventana, con un EditText, en donde agregamos:
com.google.android.gms:play-services-maps
Seleccionamos la ultima version que nos aparesca y damos click en ok, se devuelve a la anterior ventana, damos en Apply y luego en ok.

Verificamos que en el build.gradle se encuentra agregada la nueva dependencia
