Pasar informacion por Intent

Para pasar información, se coloca el método putExtra(CLAVE, variable) en el objeto Intent del activity que envía y en el activity que recibe, los métodos getIntent() y getStringExtra() para un objeto Intent .
A continuación se presentan los dos códigos por separado, se debe tener presente que la Clave se debe declarar como una constante String

  • Activity que Envia

Este metodo es llamado por onClick 

                public static final String EXTRA_LATITUD = "com.tracking.justlocation";
                *******************
                public void irSecond(View view) {
                  updateGPS();
                  Intent intent = new Intent(this, segundoActivity.class);
                  intent.putExtra(EXTRA_LATITUD, latitudx1);
                  startActivity(intent);
              }

  • Activity que Recibe

En el activity que recibe los métodos son colocados en el onCreate, para que se ejecuten al iniciar el activity, para este ejemplo son inmediatamente ubicados en los textView 

                protected void onCreate(Bundle savedInstanceState) {
                  super.onCreate(savedInstanceState);
                  setContentView(R.layout.activity_segundo);
          
                  lat = findViewById(R.id.textView);
                  lon = findViewById(R.id.textView2);
          
                  Intent intent = getIntent();
                  String data = intent.getStringExtra(MainActivity.EXTRA_LATITUD);
                  lat.setText(data);
              }
              }

  • Enviar varios datos

En el activity que envía se coloca un método putExtra() para cada dato, enlazándolo al mismo objeto Intent, diferenciándose en la CLAVE y lo que envían, mientras en el activity que recibe se coloca un método getStrinExtra para cada variable que se recupera 

                public void irSecond(View view) {
                  updateGPS();
                  Intent intent = new Intent(this, segundoActivity.class);
          
                  intent.putExtra(EXTRA_LATITUD, latitudx1);
                  intent.putExtra(EXTRA_LONGITUD, longitudx1);
          
                  startActivity(intent);
              }
              }

                protected void onCreate(Bundle savedInstanceState) {
                  super.onCreate(savedInstanceState);
                  setContentView(R.layout.activity_segundo);
          
                  lat = findViewById(R.id.textView);
                  lon = findViewById(R.id.textView2);
          
                  Intent intent = getIntent();
                  String datalat = intent.getStringExtra(MainActivity.EXTRA_LATITUD);
                  String datalon = intent.getStringExtra(MainActivity.EXTRA_LONGITUD);
                  lat.setText(datalat);
                  lon.setText(datalon);
              }

Validacion EditText

La siguiente validacion es para un EditText, y verfica, primero que las casillas no esten vacias, no sean iguales y tengan una cantidad limite de caracteres, se plantea como dos metodos, en donde el primero evalua la condicion y el siguiente establece los mensaje de negacion y el acceso a la funcionalidad del boton. 

                
                  public String validar(){

                    String validacion ="bien";
                    String campo1 = et_nombreProyecto.getText().toString();
                    String campo2 = et_IdTransecto.getText().toString();
                    if(campo1.isEmpty() || campo2.isEmpty()){
                        validacion = "vacio";
                    }else if (campo1.equals(campo2)){
                       validacion = "iguales";
                    } else if (campo1.length() > 10 || campo2.length() > 10){
                    validacion = "largo";
                    }
                return validacion;
                }

                public void onClickIniciar(View view) {
                  switch (validar()){
                      case "vacio":
                          Toast.makeText(this, "No pueden estar vacios", Toast.LENGTH_LONG).show();
                         break;
                      case "iguales":
                          Toast.makeText(this, "No pueden ser iguales", Toast.LENGTH_LONG).show();
                          et_IdTransecto.setText("");
                          break;
                      case "largo":
                          Toast.makeText(this, "no pueden ser tan grandes", Toast.LENGTH_LONG).show();
                          et_nombreProyecto.setText("");
                          et_IdTransecto.setText("");
                          break;
                      default:   iniciarProyecto();
                  }
           
               }
