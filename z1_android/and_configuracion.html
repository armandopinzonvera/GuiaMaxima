Cambiar nombre Proyecto

Se puede iniciar con un proyecto clonado de GitHub o copiando un proyecto que ya exista. Lo primero que hacemos es colocarle el nuevo nombre a la carpeta en donde queremos hacer el cambio

Abrimos AnSt sin dejar que carguen los proyectos de la memoria, como la nueva copia no se ha utilizado, no aparece en el listado de la izquierda; así que abrimos Open an existing Android Studio proyect, con lo cual se abre el directorio, ubicamos la carpeta carpeta a la que recién asignamos el nuevo nombre y damos click en ok

Al abrir el directorio de este proyecto, se observa como aún conserva el antiguo nombre en algunas carpetas y la dirección; Iniciamos separando las carpetas que almacenan los archivos .java: seleccionando la primera carpeta, abriendo el engranaje de opciones y deseleccionando Compact Middle Package, con lo cual la carpeta inicial se separa en tres, en donde cada nombre corresponde con el inicial, separado por el punto

Damos Click derecho en la carpeta que queremos tenga el nuevo nombre del proyecto, seleccionamos refactor, luego rename

Aparecen una ventana de advertencia (Warning) y damos clic en Rename package.

Aparece una nueva ventana, Rename, se coloca el nuevo nombre en la primera pestaña de opciones y se da click en Refactor.

Se abre la pantalla de find, mostrando todas las carpetas relacionadas y volvemos a dar click en Do Refactor.

Android estudio nos indica que algunos archivos tienen nombre inválidos, pero eso se corrige más adelante cuando se sincronice el gradle.

Ahora nos dirigimos al archivo Build.gradle(Module:app) y cambiamos el nombre en la línea: applicationId, teniendo presente que cada punto indica el nombre de una carpeta

Sincronizamos la información de gradle, para que la información se actualice

En la pestaña Build, seleccionamos Rebuild Project y esperamos que Android Studio actualice

En el archivo string.xml se cambia el valor del atributo ”app_name” con el correspondiente al nuevo nombre, luego en file, seleccionamos: sabe all y reiniciamos Android Studio

Spinner

Colocamos el Elemento Spiner <Spinner en el lugar donde queramos en nuestro .xml, le colocamos su etiqueta id android:id="@+id/idspinner".
Los Items que va a tener el Spinner, se crean a travez de un archivo de recursos (.xml).
Para lo cual seguimos la ruta: res/values (C|D) new/values resources file, a lo que nos abre une nueva ventana, en donde colocamos el nombre, para este caso: Spinner_opciones.xml y damos en ok.

Dentro de este archivo de recursos, generamos el Array de datos que vamos a presentar en el combo, mediante la etiqueta string-array, en donde la propiedad name="", se refiere al nombre del combo.
En el interior del string-array, colocamos cada uno de los elementos que queremos se vean, dentro de etiquetas <item> 


<?xml version="1.0" encoding="utf-8"?>
<resources>
    <string-array name="combo_especies">
        <item>especie 1</item>
        <item>especie 2</item>
        <item>especie 3</item>
        <item>especie 4</item>
        <item>especie 5</item>
    </string-array>
</resources>

Ahora colocamos la logica, en la clase (.java) del GUI, para eso hacemos la referencia, al elemento Spinner y al TextView, declarandolos y utilizando el metodo findViewById(), procedemos relacionando ese elemento del .xml del GUI con el archivo de recuersos, en donde se encuentran los items, para ello colocamos, ya sea en el metodo onCreate, o en un metodo que tengamos para ese fin:
ArrayAdapter<CharSequence> adapter = ArrayAdapter.createFromResource(this, R.array.combo_especies, android.R.layout.simple_spinner_item);
Ese adaptador se lo agregamos al elemento Spinner, que previamente habiamos declarado:
spinner_especies.setAdapter(adapter);

Procedemos colocando un evento para cuando se presiona un elemento del Spinner, mediante: el metodo setOnItemSelectedListener() a la variable spinner.
spinner_especies.setOnItemSelectedListener(new AdapterView.OnItemSelectedListener()
con lo cual se abren dos metodos sobrescritos; po rlo pronto solo colocamos un Toast, pero en el se observa, como se relacionanan los alementos de la funcion. 


    public void spinnerManager(){
        ArrayAdapter<CharSequence> adapter = ArrayAdapter.createFromResource(this,
                R.array.combo_especies, android.R.layout.simple_spinner_item);
        spinner_especies.setAdapter(adapter);

        spinner_especies.setOnItemSelectedListener(new AdapterView.OnItemSelectedListener() {
            @Override
            public void onItemSelected(AdapterView<?> parent, View view, int position, long id) {
                Toast.makeText(VistaTransecto.this,
                        "Seleccion: "+parent.getItemAtPosition(position).toString(), Toast.LENGTH_SHORT).show();
            }
            @Override
            public void onNothingSelected(AdapterView<?> parent) {
            }
        });
    }
