Habilitar Fragment

Lo primero es que debemos tener un activity tipo Fragment, extends Fragment, que sera el Fragment propiamente dicho, y un componente FrameLayout etiqueta: < FrameLayout > en el .xml del activity con el que interactua el usuario, es decir que ambos activitys tienen este FrameLayout
Entonces en el Activity que interactua el usuario, declaramos una Variable general Fragment ( Fragment fragment_mapa;) mientras en el metodo onCreate inicializamos esa variable con la clase del Fragment propiamente dicho ( fragment_mapa = new MapFragment();).
Luego llamamos los metodos:
getSupportFragmentManager().beginTransaction().add(R.id.marco_fragment, fragment_mapa).commit();
Colocando como parametros del metodo add(), la referencia al < FrameLayout > del activity con el que interactua el usuario, y la variable Fragment, que habiamos declarado. 

              
                Fragment fragment_mapa;

                @Override
                protected void onCreate(Bundle savedInstanceState) {
                    super.onCreate(savedInstanceState);
                    setContentView(R.layout.activity_vista_transecto);

                    fragment_mapa = new MapFragment();
                    getSupportFragmentManager().beginTransaction()
                                         .add(R.id.marco_fragment, fragment_mapa).commit();
                }

Fragment Mapa

Para Facilitar el trabajo se crea un Google Maps Fragment que ya trae incorporadas la vinculos y los metodos para mostrar el mapa; en la carpeta del proyecto CD/new/Fragment/Google Maps Fragment.
Faltando solo hacer los vinculos descritos en el apartado anterior, agregar la dependencia en el build.gradle y agregar la API KEY en el Manifest.
Para obtener el numero, se sigue el proceso descrito en el capitulo de Android / Map-Location de esta guia.
Para agregar la dependencia de google-play-service-maps en Project Structure File/Project Structure/Dependencies, selecionamos +/Library Dependency
a lo que se abre una nueva ventana, con un EditText, en donde agregamos:
com.google.android.gms:play-services-maps
Seleccionamos la ultima version que nos aparesca y damos click en ok, se devuelve a la anterior ventana, damos en Apply y luego en ok.

Verificamos que en el build.gradle se encuentra agregada la nueva dependencia
